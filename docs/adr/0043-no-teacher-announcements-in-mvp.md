# ADR 0043: No Teacher Announcements Or Messaging In MVP

## Status

Accepted

## Context

Teacher-to-student announcements may be useful, but messaging creates notification, moderation, delivery, and support scope. The MVP should stay focused on exam delivery and results.

## Decision

Teacher announcements/messages to enrolled students are not included in MVP.

## Consequences

Benefits:

- Keeps product scope focused.
- Avoids notification infrastructure.
- Teachers can continue using existing channels like WhatsApp/Facebook for announcements.

Future:

- Add announcements after core exam workflow is proven.
