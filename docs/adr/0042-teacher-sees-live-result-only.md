# ADR 0042: Teacher Sees Live Result Only In MVP

## Status

Accepted

## Context

Mock attempts are practice. Showing all mock attempt history to teachers may increase scope and create pressure around practice behavior.

## Decision

In MVP, teachers see live exam results only. Students can see all of their own mock attempts.

## Consequences

Benefits:

- Keeps teacher analytics simpler.
- Protects student practice space.
- Keeps official result separate from mock behavior.

Future:

- Teacher mock visibility can be added later if teachers need coaching insight from practice attempts.
